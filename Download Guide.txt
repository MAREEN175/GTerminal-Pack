GTerminal 1.0 is a lightweight desktop simulation system built with Python and Pygame. 
It features a boot animation, draggable icons, windowed apps (Notepad, Paint, Calculator, Clock), 
a terminal, a trash bin, and file saving.

âœ… Installation Steps

1. Install the whole system each file separately

2. If you install make sure everything is on desktop

3. Group all files in 1 folder and name the folder "GTerminal"

4. IMPORTANT!!! Make sure all the files are placed in the right order
(if no system will not work properly!)

5. Install python from Microsoft Store or from the website
(recomend from Microsoft Store)

6. Install dependencies:
   pip install pygame

7. Run the system:
   python gterminal.py
